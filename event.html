TECHNOVATION


TECHNOVATION  is an initiative of the Muthoot Institute of Technology and Science Placement Cell.


The event expects interested students from top engineering colleges across Kerala. Participants will have 24 hours to build working technology and to create solutions pertaining to Industry. Participants will create concepts for software or hardware based program/applications in the categories that are specified by the industry. Some possible categories are,

  • IOT
  • Big Data
  • Mobile Application Development
  • Web Development

Challenge

Participants will create software and hardware applications that provide practical solutions in one of the following areas.

• Internet Of Things

IOT or Internet Of Things is an upcoming technology that makes use of internet to control/monitor electronic/mechanical devices, automobiles and other physical devices connected to the internet. IOTgives user the ability to control more than digital things easily through a comfortable GUI over the internet. Creating a solution for a real world problem in this field.Some sample projects are given below:

IOT based Smart Parking in busy city environment. Monitoring of parking spaces in the city. Monitoring of vibrations and material conditions in buildings, bridges and historical monuments which info is critical to government. IOT based Smart Waste Management in cities. Detection of garbage points and rubbish levels in containers to optimize the trash collection routes.

Judging Criteria

Each submission will be scored in each round based on the following criteria , with the final score being the average of the judges’ scores:

• Simplicity : Is the application simple to use and can the team explain it clearly in three sentences or less?

• Creativity : How creative was the team in developing an innovative solution for the challenge?

• Impact : Did the team create an application that can have a real and valuable impact?

• Design : Was the UX/UI intuitive and appealing?

Rules

• Fresh Code Enforced: We all start coding at the same time probably 12 noon, Saturday. This is done to keep things fair. Please don’t come in and build on top of previous projects.

• No PowerPoints: You must demo your solution in order to present. No slide-decks or power-points will be allowed. 10 minute presentation 5 minute Q&A from Judges.

• Submissions: For each software solution, it has to be submitted to the github repository and link of the same should be sent to mitstechnovation@gmail.com. Submission time is 12:00 noon Sunday unless stated otherwise.

• Code Review:Winning teams will be subject to a code-review at some point following the event or immediately before winning.

• Team Size:Teams of up to 4 humans are allowed.
